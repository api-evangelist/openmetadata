# OpenMetadata

OpenMetadata is an open-source data catalog and metadata platform providing a comprehensive REST API for managing tables, dashboards, pipelines, data quality, lineage, and governance policies. It serves as an open context layer for data and AI, unifying technical metadata, business semantics, and organizational memory into a unified graph for both human and AI decision-making.
